# Mpc Ahu Neural Network

## Description

Repo for the work on a Neural Network approximation of the MPC solution controlling separately both the HVAC_SR1_PLC1 and HVAC_SR1_PLC2.

For more information about the MPC solution for the HVAC system consider taking a look at [https://mpc-cv.docs.cern.ch/](https://mpc-cv.docs.cern.ch/).
